# Mini Restaurant Management App (Role Based)

## Tech Stack
- React
- React Router
- Context API
- LocalStorage

## Features
- Role-based login (Admin / Customer)
- Protected routes
- Admin can add, update, delete restaurants
- Customer can only view restaurants
- Search and filters
- Data persisted using localStorage
